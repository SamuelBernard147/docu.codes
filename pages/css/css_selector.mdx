
# Selector 

Selektor CSS digunakan untuk memilih konten yang ingin diberi style . Selector adalah bagian dari kumpulan aturan CSS. Selektor CSS memilih elemen HTML sesuai dengan id, class, atribut, dll.

Ada beberapa jenis penyeleksi di CSS.

* Selector CSS mengunakan Elemen HTML
* Selector CSS mengunakan atribut Id
* Selector CSS mengunakan atribut Class


## Selector mengunakan Elemen HTML
Pemilih elemen memilih elemen HTML berdasarkan nama elemen.

## Selector mengunakan atribut Id
Selector mengunakan atribut id dari elemen HTML untuk memilih elemen tertentu. Sebuah id selalu unik di dalam halaman sehingga dipilih untuk memilih satu elemen unik.

Itu ditulis dengan karakter hash (#), diikuti oleh id elemen.

Mari kita ambil contoh dengan id "para1".
<div >
<Trycode code_source="https://www.w3schools.com/html/tryit.asp?filename=tryhtml_css_internal">
    ```css

// CSS
#header_id {
  text-align: center;
  color: red;
}

// HTML
<p id="header_id"> Selamat Datang! </p>   
    ```
    </Trycode>
</div>
