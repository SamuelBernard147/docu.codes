
# Syntax CSS

Kumpulan aturan CSS berisi pemilih dan blok deklarasi.

![css_sintaks!](/css/css_sintaks.png "css_sintaks")


* **Selector** : Selector menunjukkan elemen HTML yang ingin diberi style. element bisa berupa tag apa saja seperti ``<h1>``, ``<p>`` dll.

* **Declaration Block**:Declaration Block berisi satu atau lebih sintaks CSS yang dipisahkan oleh titik koma. Setiap sintaks menyertakan nama properti CSS dan nilai
