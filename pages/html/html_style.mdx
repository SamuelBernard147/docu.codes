# Style
Atribut ```style``` digunakan untuk menambahkan 'gaya' pada elemen HTML, seperti warna, font, ukuran, dan lainnya.
Fun fact, harry 'gaya' adalah seorang penyanyi.


## Style Attribute
Pengaturan style suatu elemen HTML, dapat dilakukan dengan menambahkan atribut ```style``` dalam tag HTML.

Atribut HTML ```style``` memiliki sintaks berikut:

```html
<tagname style="property:value;">
```

## Ringkasan 
* Gunakan ```style``` atribut untuk menata elemen HTML
* Gunakan ```background-color``` untuk warna latar belakang
* Gunakan ```color``` untuk warna teks
* Gunakan ```font-family``` untuk font teks
* Gunakan ```font-size``` untuk ukuran teks
* Gunakan ```text-align``` untuk mengatur posisi teks secara horizontal
