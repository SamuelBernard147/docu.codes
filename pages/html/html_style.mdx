# Style
Atribut HTML styledigunakan untuk menambahkan gaya ke elemen, seperti warna, font, ukuran, dan lainnya.



## Style Attribute
Pengaturan style suatu elemen HTML, dapat dilakukan dengan ```style``` atribut.

Atribut HTML stylememiliki sintaks berikut:

```html
<tagname style="property:value;">

```

## Ringkasan 
* Gunakan ```style``` atribut untuk menata elemen HTML
* Gunakan ```background-color``` untuk warna latar belakang
* Gunakan ```color``` untuk warna teks
* Gunakan ```font-family``` untuk font teks
* Gunakan ```font-size``` untuk ukuran teks
* Gunakan ```text-align``` untuk perataan teks
