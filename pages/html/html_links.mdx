# Link

Link atau Hyperlink adalah elemen HTML yang berfungsi menghubungkan suatu halaman web ke halaman web yang lain.

Saat Anda menggerakkan mouse di atas tautan, panah mouse akan berubah menjadi tangan kecil.

Untuk membuat hyperlink kita dapat mengunakan Tag ```<a>```. kemudian tag ini harus memiliki atribut href untuk menentukan alamat URL tujuan dari link



import Trycode from '../../component/trycode'

<div >
<Trycode code_source="https://www.w3schools.com/html/tryit.asp?filename=tryhtml_intro">
    ```html
        <a href="https://www.google.co.id/">Pindah ke google</a>
    ```
    </Trycode>
</div>

Secara default, tautan akan muncul sebagai berikut di semua browser:

* Tautan yang belum dikunjungi digarisbawahi dan berwarna biru
* Tautan yang dikunjungi digarisbawahi dan berwarna ungu
* Tautan aktif digarisbawahi dan merah

## Jenis-jenis Link pada HTML
Link pada HTML dapat dibagi menjadi dua kelompok berdasarkan alamat URL yang dituju

* **Internal Link**: adalah link yang menuju ke domain atau halaman web itu sendiri;

![Petanikode](https://www.petanikode.com/img/html/link/internal-link.png)

**Contoh**\
pastikan didalem folder yang sama terdapat file dengan nama ```about.html```
<div >
<Trycode code_source="https://www.w3schools.com/html/tryit.asp?filename=tryhtml_intro">
    ```html
        <a href="about.html">Pindah ke About</a>
    ```
    </Trycode>
</div>



* **External Link**: adalah link yang menuju domain lain.

![Petanikode](https://www.petanikode.com/img/html/link/eksternal-link.png)

**Contoh**
<div >
<Trycode code_source="https://www.w3schools.com/html/tryit.asp?filename=tryhtml_intro">
    ```html
                <a href="https://www.google.co.id/">Pindah ke google</a>

    ```
    </Trycode>
</div>
