# Text Formatting
HTML berisi beberapa elemen untuk mendefinisikan teks dengan arti khusus.

## Formatting Elements
Elemen pemformatan dirancang untuk menampilkan jenis teks khusus:

* ```<b>``` = Teks tebal
* ```<strong>``` = Teks penting
* ```<i>``` = Teks miring
* ```<em>``` = Teks yang ditekankan
* ```<mark>``` = Teks yang ditandai
* ```<small>``` = Teks yang lebih kecil
* ```<del>``` = Teks yang dihapus
* ```<ins>``` = Teks yang disisipkan
* ```<sub>``` = Teks subskrip
* ```<sup>``` = Teks superskrip

## Elemen HTML ```<i>``` dan ```<em>```
Elemen HTML ```<i>```mendefinisikan bagian teks dalam suara atau suasana hati alternatif. Konten di dalamnya biasanya ditampilkan dalam huruf miring.

**Tip**: Tag ```<i>```sering digunakan untuk menunjukkan istilah teknis, frasa dari bahasa lain, pemikiran, nama kapal, dll.

<div >
<Trycode code_source="https://www.w3schools.com/html/tryit.asp?filename=tryhtml_formatting_i">
    ```html
<i>This text is italic</i>

    ```
    </Trycode>
</div>

Elemen HTML ```<em>``` mendefinisikan teks yang ditekankan. Konten di dalamnya biasanya ditampilkan dalam huruf miring.

**Tip** : Pembaca layar akan mengucapkan kata-kata ```<em>``` dengan penekanan, menggunakan tekanan verbal.

<div >

    </Trycode>

##  Elemen HTML ```<ins>```
Elemen HTML ```<ins>``` mendefinisikan teks yang telah dimasukkan ke dalam dokumen. Browser biasanya akan menggarisbawahi teks yang disisipkan:
<div >

##  Elemen HTML ```<sub>```
Elemen HTML ```<sub>``` mendefinisikan teks subscript. Teks subskrip muncul setengah karakter di bawah garis normal, dan terkadang ditampilkan dalam font yang lebih kecil. Teks subscript dapat digunakan untuk rumus kimia, seperti H <sub>2</sub> O:

<div >
<Trycode code_source="https://www.w3schools.com/html/tryit.asp?filename=tryhtml_formatting_del_ins">
```html
<p>My favorite color is <del>blue</del> <ins>red</ins>.</p>
```
    </Trycode>
</div>

##  Elemen HTML ```<sup>```
```<sup>``` Elemen HTML mendefinisikan teks superscript. Teks superskrip muncul setengah karakter di atas garis normal, dan terkadang ditampilkan dalam font yang lebih kecil. Teks superscript dapat digunakan untuk catatan kaki, seperti WWW <sup>[1]</sup> :
<div >
<Trycode code_source="https://www.w3schools.com/html/tryit.asp?filename=tryhtml_formatting_del_ins">
```html
<p>My favorite color is <del>blue</del> <ins>red</ins>.</p>
```
    </Trycode>
</div>

## Ringkasan


* ```<b>``` Mendefinisikan bold text
* ```<em>``` Mendefinisikan emphasized text 
* ```<i>``` Mendefinisikan a part of text in an alternate voice or mood
* ```<small>``` Mendefinisikan smaller text
* ```<strong>``` Mendefinisikan important text
* ```<sub>``` Mendefinisikan subscripted text
* ```<sup>``` Mendefinisikan superscripted text
* ```<ins>``` Mendefinisikan inserted text
* ```<del>``` Mendefinisikan deleted text
* ```<mark>``` Mendefinisikan marked/highlighted text
