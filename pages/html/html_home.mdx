# HTML Tutorial

## Belajar Mudah dengan HTML

Dengan editor "Try it Yourself" yang di sediakan W3School, Anda dapat mengedit kode HTML dan melihat hasilnya:


import Trycode from 'nextra-theme-docs/trycode'

<div >
<Trycode code_source="https://www.w3schools.com/html/tryit.asp?filename=tryhtml_intro">
    ```markdown
    <!DOCTYPE html>
    <html>
        <head>
            <title>Page Title</title>
        </head>
        <body>
            <h1>This is a Heading</h1>
            <p>This is a paragraph.</p>
        </body>
    </html>
    ```
    </Trycode>
</div>
