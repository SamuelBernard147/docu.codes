# Tabel
Tabel dapat memungkinkan kita untuk mengatur data ke dalam baris dan kolom.


## Membuat Tabel
Untuk membuat tabel kita membutuhkan beberapa tag yang wajib ada:

* Tag ``<table>`` untuk membungkus tabelnya
* Tag ``<tr>`` (tabel row) untuk membuat baris
* Tag ``<td>`` (table data) untuk membuat sel

```html
    <table>
        <tr>
            <td>Nama</td>
            <td>Umur</td>
        </tr>
        <tr>
            <td>Multatuli</td>
            <td>45 thn</td>
        </tr>
        <tr>
            <td>Albert</td>
            <td>35 thn</td>
        </tr>
    </table>
```


## Table Row
Setiap baris tabel dimulai dengan ``<tr>`` dan diakhiri dengan ``</tr>`` tag.


```html
<table>
  <tr>
    <td>Emil</td>
    <td>Tobias</td>
    <td>Linus</td>
  </tr>
  <tr>
    <td>16</td>
    <td>14</td>
    <td>10</td>
  </tr>
</table>
```

## Tabel cell
Setiap sel tabel didefinisikan oleh ``<td>`` dan ``</td>`` tag.

Segala sesuatu di antara ``<td>``dan ``</td>``adalah konten sel tabel.

```html
<table>
  <tr>
    <td>Emil</td>
    <td>Tobias</td>
    <td>Linus</td>
  </tr>
</table>
```

## Tabel Header
Terkadang Anda ingin cell Anda menjadi title dari sebuah row, dalam kasus tersebut gunakan ``<th>`` tag alih-alih ``<td>`` tag:

```html
<table>
  <tr>
    <th>Person 1</th>
    <th>Person 2</th>
    <th>Person 3</th>
  </tr>
  <tr>
    <td>Emil</td>
    <td>Tobias</td>
    <td>Linus</td>
  </tr>
  <tr>
    <td>16</td>
    <td>14</td>
    <td>10</td>
  </tr>
</table>
```
