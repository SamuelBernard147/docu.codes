# List
List memungkinkan pengembang web untuk mengelompokkan satu set item terkait dalam satu kelompok


## Elemen List
Untuk membuat list pada HTML. kita dapat mengunakan salah satu dari tiga macam jenis list yang telah disediakan oleh HTML:

* **Ordered** List adalah list yang terurut;
* **Unordered** List adalah list yang tak terurut;
* **Descriptiona** List adalah list yang berisi definisi.

## Description List 

Description List adalah list yang berisi deksripsi atau penjelasan dari sesuatu.

Ada tiga tag yang digunakan untuk membuat description list:

* ``<dl>`` (description list) tag untuk memulai description list;
* ``<dt>`` (description term) tag untuk membuat kata yang akan dideskripsikan;
* ``<dd>`` (description description) tag untuk membuat penjelasan dari kata.


<div>
<Trycode code_source="https://www.w3schools.com/html/tryit.asp?filename=tryhtml_basic_headings">
    ```html
<dl>
  <dt>Coffee</dt>
  <dd>- black hot drink</dd>
  <dt>Milk</dt>
  <dd>- white cold drink</dd>
</dl>
    ```
    </Trycode>
</div>
