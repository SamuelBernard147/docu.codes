# Komentar 
Komentar adalah elemen yang akan diabaikan oleh browser. Ia tidak akan ditampilkan di dalam web, tetapi dapat membantu mendokumentasikan kode sumber HTML Anda.
## Tag Komentar

Anda dapat menambahkan komentar ke sumber HTML Anda dengan menggunakan sintaks berikut:

    ```html
<!-- isi komentar -->
    ```
> **Perhatikan** bahwa ada tanda seru (!) di tag awal, tetapi tidak di tag akhir.

Dengan komentar Anda dapat menempatkan pemberitahuan dan pengingat dalam kode HTML Anda:

import Trycode from '../../component/trycode'

<div >
<Trycode code_source="https://www.w3schools.com/html/tryit.asp?filename=tryhtml_comment">
    ```html
<!-- ini adalah paragraf -->
<p>This is a paragraph.</p>

<!-- ingatkan saya untuk memberi Heading -->
    ```
    </Trycode>
</div>

## Sembunyikan Konten
Komentar dapat digunakan untuk menyembunyikan konten HTML untuk sementara:

<div >
<Trycode code_source="https://www.w3schools.com/html/tryit.asp?filename=tryhtml_comment_hide">
    ```html
<p>Ini paragraf.</p>

<!-- <p>Ini paragraf lain </p> -->

<p>Ini juga paragraf.</p>
    ```
    </Trycode>
</div>
